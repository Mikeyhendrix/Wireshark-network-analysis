# Wireshark Network Analysis Project

## Overview
This project demonstrates capturing and analyzing network traffic using Wireshark. DNS queries and TCP traffic were examined to understand how devices communicate over a network.

## Key Findings
- Identified DNS queries to domains such as ESPN CDN services
- Analyzed TCP traffic over HTTPS (port 443)
- Examined packet structure including IP addresses, ports, and flags

## Tools Used
- Wireshark
